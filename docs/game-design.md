# Game Design: 破産防衛都市

## コンセプト

破産防衛都市は、都市の財政破綻を敵として扱う防衛ゲームです。敵はモンスターではなく、債務膨張、税収減、災害復旧費、企業撤退、住民不満、インフラ老朽化です。プレイヤーは区画を守り、短期的な支出と長期的な信用を両立させます。

## プレイヤー体験

- 1 wave は 2-4 分で完了。
- 戦闘前に都市区画、政策カード、無料通貨ガチャで得た支援を編成。
- 戦闘中は下部 command dock から予算配分、区画補修、緊急政策を発動。
- 戦闘後は信用格付け、住民満足、債務残高、税収見込みが更新される。

## Core Loop

1. 都市区画を確認する。
2. 政策カードと防衛施設を編成する。
3. wave を開始する。
4. 財政 pressure を受けながら区画を守る。
5. 報酬として無料通貨、素材、信用回復を得る。
6. ガチャまたは強化で次の wave に備える。

## 主要リソース

- 信用: 0 になると破産。最重要 HP。
- 現金: wave 中の即時操作に使う。
- 債務: 借入で増え、利払い pressure を生む。
- 住民満足: 税収と protest risk に影響。
- 復興素材: 区画修復と施設強化に使う。
- 無料通貨: ガチャ、時短、低リスクな支援獲得に使う。

## 敵 Pressure

- 利払いショック: 債務が多いほど継続ダメージ。
- 税収減: 商業区画に pressure。
- 災害復旧: 複数区画を同時に削る。
- 企業撤退: 将来収入を下げる。
- 住民不満: 操作コストを上げる。
- 監査イベント: 不透明な支出を制限する。

## 区画

- 市庁舎: 信用の中核。破壊されると大幅ペナルティ。
- 商業区: 税収源。守るほど現金が増える。
- 住宅区: 住民満足の中核。
- 工業区: 素材と雇用を生むが災害に弱い。
- 港湾区: 高収益だが企業撤退 pressure を受けやすい。
- 研究区: 政策カードの cooldown 短縮。

## カード / 施設

- 緊急融資: 現金を得るが債務増。
- 支出削減: 即時コスト減、住民満足低下。
- 復興債: 災害対策に強いが利払い増。
- 中小企業支援: 商業区の税収維持。
- 住民説明会: protest risk 低下。
- インフラ更新: 老朽化 pressure を予防。

## 無料通貨ガチャ

ガチャは都市運営支援の入手手段です。PR1 では無料通貨ガチャ、排出履歴、日次上限、Turnstile gate、D1 ledger まで実装済みです。

- 排出対象: 政策カード、区画 skin、補助素材。
- 低レアでも使い道を持たせる。
- 連続ログインとステージクリアで無料通貨を配る。
- D1 ledger で grant / spend / draw を追跡する。

## 失敗条件

- 信用が 0。
- 市庁舎区画の崩壊。
- 債務が上限を超えて利払い不能。
- 住民満足が一定期間 0。

## 成功条件

- wave を耐え切る。
- 区画を一定数以上維持。
- 信用格付けを規定値以上に保つ。
- 特別ステージでは債務を一定値以下に抑える。

## PR1 Complete Scope

- 世界観と core loop。
- 都市マップ生成、区画 pressure、wave 進行。
- ゲスト / ログイン / D1 同期。
- 無料通貨ガチャ、排出履歴、ledger。
- スマホ UI、command dock、カード編成。
- asset manifest と Codex GPT image2 / placeholder generation pipeline。
- Cloudflare runtime 契約。

## Remaining Tuning

- 報酬量、wave 圧力、債務利払いのバランス調整。
- 追加アート差し替え。
- 本番 Turnstile / D1 / R2 / KV ID の差し替え。
- ランキングやイベント運用を入れる場合の別途仕様化。
